# AI Lifestyle Advisor — Frontend

## Запуск

1. Перейдите в папку `frontend`
2. Установите зависимости:
   ```
   npm install
   ```
3. Запустите приложение:
   ```
   npm start
   ```

## Структура
- `src/components` — основные компоненты интерфейса
- `src/styles` — стили
- `src/api.js` — функции для работы с backend
